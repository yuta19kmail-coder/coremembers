# CoreMembers 本番化セットアップ手順（coremembers.kobayashi-motors.com）

> 🎯 v0.1.0（Phase1：社員名簿＋組織図＋場所マスター）。アプリ側のコードは本番接続済み。
> あとは MHS と同じ要領で **Firebaseコンソール・DNS・OAuth・デプロイ** を設定するだけ。
> プロジェクトは **carflow-9d500** に相乗り。⚠ A→B→C→D が終わるまでログインは通りません。

## A. Hosting にサイト作成
1. Firebase コンソール → **carflow-9d500** → Hosting → 「別のサイトを追加」→ サイトID = **`coremembers-km`**

## B. 配信して表示確認（PowerShell・`D:\アプリ開発\CoreMembers`）
```powershell
cd D:\アプリ開発\CoreMembers
git init ; git add -A ; git commit -m "CoreMembers v0.1.0 初期"
firebase target:apply hosting coremembers coremembers-km
firebase deploy --only hosting:coremembers
```
→ `https://coremembers-km.web.app` でログイン画面が出ればOK（この時点ではまだログイン不可で正常）。
（GitHubに上げるなら：リポジトリ `coremembers` を作成して `git remote add origin … ; git push -u origin main`）

## C. 独自ドメイン coremembers.kobayashi-motors.com 割り当て
1. Hosting → **coremembers-km** → 「カスタムドメインを追加」→ `coremembers.kobayashi-motors.com`
2. 表示された **CNAME**（→ coremembers-km.web.app）を、carflow/mhs と同じDNS管理画面に追加（ホスト名＝`coremembers`）
3. 反映＋SSL発行を待つ

## D. ログイン承認（これが無いと不可）
1. Firebase → Authentication → Settings → **承認済みドメイン** に `coremembers.kobayashi-motors.com` 追加
2. Google Cloud Console → 認証情報 → OAuth2.0クライアント（ウェブ）→ **承認済みリダイレクトURI** に
   `https://coremembers.kobayashi-motors.com/__/auth/handler` 追加

## E. Firestore ルール
- `D:\アプリ開発\CarFlow\carflow\firestore.rules` に **coreMembers / coreDepts / coreLocations を追加済み**（既存は無変更）。
- Firebase → Firestore → ルール に firestore.rules 全文を貼って「公開」（または CarFlow で `firebase deploy --only firestore:rules`）。

## F. 初期セットアップ（アプリ内・管理者のゆうたで）
1. `https://coremembers.kobayashi-motors.com` を開く → Google ログイン（管理者）
2. **📍場所マスター**：陸運局A/B・部品商・本社 等を登録（方面も入れる）
3. **🏢組織**：整備部 > 1課/2課、受付、車販、デザイン 等を登録
4. **👥社員名簿 → 📥 portalMembersから取り込み**：CoreFlow名簿の人を一括取り込み
5. 各社員を開いて 所属（課）・雇用形態・運転スキル・行ける場所 を調整
6. ログインしないバイト等は **＋新規社員** で追加（ログインアカウントは「なし」）

## 今後の更新
- `cd D:\アプリ開発\CoreMembers` → `firebase deploy --only hosting:coremembers`
- 他アプリのランチャーから CoreMembers へ飛べるように、各 `launcher.js` の coremembers url を `https://coremembers.kobayashi-motors.com` に更新（MHSの時と同じ要領・急がない）

## データ保存先（Firestore・carflow-9d500 相乗り）
- 社員：`companies/kobayashi_motors/coreMembers/{id}`（portalMemberId で portalMembers と相互リンク）
- 組織：`coreDepts/{id}` ／ 場所：`coreLocations/{id}`
- 入室名簿：`portalMembers`（CoreFlow と共用・読むだけ）
